Extraction Scripts
------------------

The extraction scripts are located under the *scripts* folder. They are responsible for obtaining and organizing data into useful formats so that the analysis scripts can be executed over such data.

spot_datax.py
*************

`spot_datax.py` is responsible for data about tracks through the Spotify API. Such task is done with the Spotipy library and yields the `spot_data.csv` file under *results*.
